You can change the default gap by setting one of the following Sass properties:

  • $grid-gap - Default: 1.5rem
  • $grid-column-gap - Default: $grid-gap
  • $grid-row-gap - Default: $grid-gap
{% include elements/anchor.html name="Gapless" %}

You can specify a gapless grid by using the following modifiers:

  • is-column-gapless - No gap on columns
  • is-row-gapless - No gap on rows
  • is-gapless - No gap for the whole grid
