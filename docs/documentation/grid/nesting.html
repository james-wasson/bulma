{% include elements/anchor.html name="Nested Children" %}

You can nest grid children inside one another by using the has-grid-children class:

{{ grid_child_nesting_example }}
{% highlight html %}{{ grid_child_nesting_snippet }}{% endhighlight %}

Nested grid children have the same properties as a regular grid child. Which means you can use all of the same size/control classes such as:

  • is-left-* e.g. is-left-1
  • is-height-* e.g. is-height-3

By default Bulma supports nesting up to 3 deep.
