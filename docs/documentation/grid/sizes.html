You can specify the grid size by using the following modifiers:

  • has-column-count-* e.g. has-column-count-1
  • has-row-count-* e.g. has-row-count-3

By default Bulma supports column and row counts up to 12. You can change this default by setting $grid-column-row-count to a larger value.

The grid has 3 row size modifiers:

  • has-rows-equal - All rows have equal size
  • has-rows-fullheight - All rows are fit inside 100vh
  • has-rows-fullheight-with-nav - All rows are fit inside 100vh - $navbar-height

The following responsive modifiers are supported for the grid:

  • has-column-count-*-mobile
  • has-column-count-*-tablet
  • has-column-count-*-touch
  • has-column-count-*-desktop
  • has-column-count-*-widescreen
  • has-column-count-*-fullhd
